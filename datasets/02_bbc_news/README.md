# Project 02: BBC News Classification

## Problem Statement
Given the text of a news article, categorize it into one of 5 distinct topics: **Business, Tech, Politics, Sport, or Entertainment**.

## Dataset
- **Name:** BBC News Dataset
- **Task:** Multi-class Text Categorization
- **Classes:** 5

## Learning Goals
- Multi-class classification strategies
- Stemming vs. Lemmatization
- Vocabulary size tuning
- Evaluating performance on balanced categorical data using traditional classifiers like Logistic Regression.
