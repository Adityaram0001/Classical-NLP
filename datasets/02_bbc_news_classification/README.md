# BBC News Classification Dataset

## Overview
A dataset of 2,225 BBC News articles categorized into 5 classes: business, entertainment, politics, sport, and tech.

## Learning Goals
- Multi-class text categorization
- Understanding the difference in vocabulary reduction between Stemming and Lemmatization
- Vocabulary size tuning (`max_features`, `min_df`)
- Evaluating multi-class performance (Macro vs Micro F1)

## Data Source
The data is downloaded directly into `data/raw/dataset.csv`.
