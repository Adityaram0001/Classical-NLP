# Project 04: A Million News Headlines (Topic Modeling)

## Problem Statement
Given a massive corpus of short news headlines without any labels, **algorithmically discover and extract the hidden thematic topics** running through the news over time.

## Dataset
- **Name:** A Million News Headlines Dataset (Sampled to 100k)
- **Task:** Unsupervised Learning (Topic Modeling)
- **Classes:** Unlabeled

## Learning Goals
- Transitioning to Unsupervised Learning.
- Highly aggressive noise reduction (strict stopwords, length filtering).
- Latent Dirichlet Allocation (LDA) using Bag-of-Words.
- Non-Negative Matrix Factorization (NMF) using TF-IDF.
- Document-topic distribution mapping and understanding which algorithm handles short text better.
