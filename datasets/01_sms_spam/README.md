# Project 01: SMS Spam Collection

## Problem Statement
Given the raw text of an SMS message, classify whether it is **Spam** (unsolicited promotional material) or **Ham** (a legitimate message).

## Dataset
- **Name:** SMS Spam Collection Dataset
- **Task:** Binary Text Classification
- **Classes:** 2 (Spam, Ham)

## Learning Goals
- Text cleaning (Regex, Punctuation removal, Lowercasing)
- Stopword removal and Tokenization
- Bag-of-Words (CountVectorizer) vs TF-IDF vectorization
- Building a classical Naive Bayes classifier.
